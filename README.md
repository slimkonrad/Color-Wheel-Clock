# Color Wheel Clock 🎨🕰️

### Overview
This is a mobile application developed for our class project at **CSUF**. The goal is to move away from traditional digital/analog clock faces and instead represent time through the visual spectrum of a color wheel.

### How It Works
Instead of reading numbers, the user perceives time through shifting hues.
* **The Logic:** We map the standard time units (Hours, Minutes, Seconds) to the **360 degrees** of the HSL (Hue, Saturation, Lightness) color model.
* **Visuals:** As time passes, the UI rotates through the color spectrum. For example, the hue might shift by 6 degrees every second to complete a full rotation every minute.



### Key Features
* **Real-time Updates:** The UI refreshes every second to ensure a smooth color transition.
* **Minimalist Interface:** A clean design that focuses entirely on the color experience.
* **SwiftUI Core:** Built using modern declarative syntax for efficient animations.
